# SPELAB 部署到腾讯云

把本机项目部署到腾讯云 CVM 后，可通过公网 IP 访问多站点爬虫控制台、商品列表等。

---

## 一、你需要在腾讯云完成的

1. **购买一台云服务器（CVM）**
   - 地域：选离你近的即可
   - 系统：**Ubuntu 22.04** 或 **CentOS 7/8**（推荐 Ubuntu 22.04）
   - 配置：1 核 2G 即可跑；若经常多站同时爬可 2 核 4G
   - 安全组：放行 **22（SSH）**、**5001（本应用）** 或你打算用的端口（如 80）

2. **拿到并保管好**
   - 公网 IP：如 `43.xxx.xxx.xxx`
   - SSH 登录方式：密码 或 密钥（用户名一般为 `ubuntu` 或 `root`）

---

## 二、在你本机（Mac）上做的

### 1. 把项目打成压缩包（或用 git）

在 **本机** 项目根目录执行（不要包含 venv 和 data 里的大库，可选）：

```bash
cd /Users/fengchangrui/Desktop/cursor
tar --exclude='SPELAB/.venv' --exclude='SPELAB/data/sites/*/db.sqlite' -czvf SPELAB.tar.gz SPELAB
```

或用 git（若项目已在 git 仓库）：

```bash
cd /Users/fengchangrui/Desktop/cursor/SPELAB
git init && git add . && git commit -m "deploy"
# 然后推送到 GitHub/Gitee，在服务器上 git clone
```

### 2. 用 SSH 登录服务器

```bash
ssh ubuntu@你的公网IP
# 或 root@你的公网IP
# 按提示输入密码（或使用密钥）
```

## 三、在服务器上执行的命令（按顺序复制执行）

下面以 **Ubuntu 22.04**、项目放在 `/opt/spelab`、用 **5001** 端口为例；若用 CentOS，把 `apt` 换成 `yum`，包名可能略有不同。

### 1. 安装 Python 3 和 pip

```bash
sudo apt update
sudo apt install -y python3 python3-pip python3-venv
```

### 2. 上传项目到服务器

**方式 A：本机用 scp 上传压缩包**

在本机（新开一个终端）执行：

```bash
scp /Users/fengchangrui/Desktop/cursor/SPELAB.tar.gz ubuntu@你的公网IP:/tmp/
```

然后在服务器上：

```bash
sudo mkdir -p /opt/spelab
sudo tar -xzf /tmp/SPELAB.tar.gz -C /opt/spelab --strip-components=1
sudo chown -R $USER:$USER /opt/spelab
```

**方式 B：在服务器上 git clone（若你已推送到 GitHub/Gitee）**

```bash
sudo mkdir -p /opt/spelab
sudo chown $USER:$USER /opt/spelab
cd /opt/spelab
git clone 你的仓库地址 .
```

### 3. 创建虚拟环境并安装依赖

```bash
cd /opt/spelab
python3 -m venv .venv
source .venv/bin/activate
pip install --upgrade pip
pip install -r requirements-prod.txt
```

### 4. 测试运行（前台，确认能跑再关掉）

```bash
cd /opt/spelab
source .venv/bin/activate
python3 -m server.app
```

在浏览器访问：`http://你的公网IP:5001`。  
确认能打开后，在服务器终端按 `Ctrl+C` 停掉。

### 5. 用 systemd 常驻运行（推荐）

```bash
sudo nano /etc/systemd/system/spelab.service
```

写入下面内容（路径和端口可按需改）：

```ini
[Unit]
Description=SPELAB Multi-Site Crawler
After=network.target

[Service]
Type=simple
User=ubuntu
WorkingDirectory=/opt/spelab
Environment="PATH=/opt/spelab/.venv/bin"
ExecStart=/opt/spelab/.venv/bin/gunicorn -w 2 -b 0.0.0.0:5001 server.app:app
Restart=always
RestartSec=5

[Install]
WantedBy=multi-user.target
```

保存退出（`Ctrl+O` 回车，`Ctrl+X`），然后执行：

```bash
sudo systemctl daemon-reload
sudo systemctl enable spelab
sudo systemctl start spelab
sudo systemctl status spelab
```

看到 `active (running)` 即表示已常驻。之后访问：`http://你的公网IP:5001`。

### 6. 查看日志（排查问题时用）

```bash
journalctl -u spelab -f
# 或看应用自己的 server.log
tail -f /opt/spelab/server.log
```

## 四、可选：用 Nginx 做反向代理并开 80 端口

若希望用 `http://你的公网IP` 访问（不加 :5001），可在同一台机装 Nginx，把 80 转到本机 5001：

```bash
sudo apt install -y nginx
sudo nano /etc/nginx/sites-available/spelab
```

写入：

```nginx
server {
    listen 80;
    server_name _;
    location / {
        proxy_pass http://127.0.0.1:5001;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
    }
}
```

然后：

```bash
sudo ln -sf /etc/nginx/sites-available/spelab /etc/nginx/sites-enabled/
sudo nginx -t && sudo systemctl reload nginx
```

安全组放行 **80** 后，用 `http://你的公网IP` 访问即可。

## 五、安全与维护建议

- **不要**把腾讯云控制台或 SSH 的账号密码发给他人；密码只在你自己机器上输入。
- 若开放 22 端口，建议改用 **密钥登录** 并关闭密码登录。
- 环境变量（如 `SCRAPER_API_KEY`、`EGR_PROXY_URL`）可在 `/opt/spelab/.env` 里配置，不要提交到公开仓库。
- 数据在 `/opt/spelab/data/`，建议定期在服务器上备份或同步到对象存储。

## 六、你只需要做的几件事

1. 腾讯云买好 CVM、放行 22 和 5001（或 80）。  
2. 本机用 `scp` 或 `git` 把项目弄到服务器（见第二节）。  
3. 在服务器上按 **第三节** 依次执行命令。  

**我不能替你登录服务器或使用你的账号密码**，但按上面做即可完成部署。若某一步报错，把**完整命令**和**终端报错内容**发给我，我可以帮你逐条排查。
